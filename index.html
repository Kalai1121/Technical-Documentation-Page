Introduction

Python is a high-level, interpreted programming language known for its readability and versatility. It supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

Python is widely used in web development, data analysis, artificial intelligence, scientific computing, and more. It boasts a comprehensive standard library and a vibrant community.

What you should already know

This guide assumes you have the following basic background:

  • A general understanding of programming concepts.
  • Familiarity with basic command line operations.
  • No prior knowledge of Python is required, but it may help.
Python and Other Languages

Python is often compared to languages like Java and C++. While Java is statically typed and requires compilation, Python is dynamically typed and interpreted, which often leads to quicker development cycles.

Unlike C++, Python emphasizes readability and simplicity, making it easier for beginners to learn and use effectively.

Hello world
To get started with Python, you can write your first "Hello world" program: print("Hello, World!")

Run this code in your Python environment to see the output!

Variables

Variables in Python are used to store data values. Python has no command for declaring a variable; you simply assign a value to a variable name:

x = 5

In Python, variable names must start with a letter or an underscore (_) and can include letters, numbers, and underscores.

Declaring variables

You can declare variables in Python without specifying their type. For example:

name = "John"
age = 30

The variable's type is inferred from the value assigned to it.

Variable scope

In Python, variable scope refers to the region of the code where a variable is defined. Variables defined inside a function are local to that function and cannot be accessed outside it.

A variable defined outside any function is considered global and can be accessed anywhere in the code.

Constants

In Python, you can define constants by using naming conventions. By convention, constant variable names are written in all uppercase letters:

PI = 3.14

While Python does not have a built-in constant type, the convention helps signal that the variable should not be changed.

Data types

The following are the basic data types in Python:

  • int: Integer values (e.g., 42)
  • float: Floating-point values (e.g., 3.14)
  • str: String values (e.g., "Hello")
  • bool: Boolean values (True or False)

Python also supports complex numbers and more advanced data structures like lists, tuples, and dictionaries.

if...else statement

Use the if statement to execute a block of code if a condition is true. Optionally, use else to execute a block if the condition is false. The syntax looks like this:

if condition:
  # execute code

Here’s an example:

if x > 0:
  print("Positive")
else:
  print("Non-positive")
while statement

A while statement repeatedly executes a block of code as long as a condition is true. Here’s the syntax:

while condition:
  # execute code

Example:

n = 0
while n < 3:
  print(n)
  n += 1
Function definitions

Functions in Python are defined using the def keyword:

def greet(name):
  print("Hello, " + name)

Call the function like this:

greet("Alice")
